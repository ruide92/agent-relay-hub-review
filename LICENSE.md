# LICENSE（许可政策声明）

> 本文件**不是**为产品选择开源许可证，而是记录已批准的 **Phase 0–4 产品许可政策**。
> 依据：已批准 V1.1（`AGENT_RELAY_HUB_PROJECT_PROPOSAL_v1.1.md` §0.9 与许可章节），批准 commit `c9cc522b4ab01008fed390c282d6bd5a816ee779`。

---

## 当前许可政策（Phase 0–4）

1. **产品源码保持私有、专有（proprietary）。**
2. **暂不承诺 Open Core**；不采用开放核心分发模式。
3. **不得擅自采用 Apache-2.0 或其他任何开源许可证发布产品源码。**
4. **Phase 7 前重新进行商业模式和许可裁决**：许可证与商业模式的最终裁定属 Phase 7（见 `ROADMAP.md`），在此之前不得对外开源或商业发布产品源码。
5. 受限许可组件（GPL / AGPL / SSPL / BUSL / Sustainable Use License 等）未经逐项法律兼容审查与授权，不得随产品捆绑分发。

---

## 声明范围与边界

- 本文件是**当前许可政策声明**，**不代表已经完成所有第三方组件法律审查**。
- 第三方组件的逐项审查登记见 `THIRD_PARTY_LICENSES.md`；软件物料清单政策见 `SBOM_POLICY.md`。
- 第三方组件必须按**实际使用版本、具体文件/包路径、对应许可证文本**逐项审查；**不得使用 fair-code、source-available 等营销或类别标签代替正式许可证名称**。
